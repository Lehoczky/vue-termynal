# Contributing

# Initial setup

First forkopen in new window the Vue-Termynal repository then clone and build your fork locally. Run the following commands:

git clone https://github.com/your-username/vue-termynal.git
npm install

# Running your code

During development you can test your work in the playground folder. You can start the app inside it by running:

npm run dev
1

# Lint errors

The project uses ESlintopen in new window, stylelintopen in new window and prettieropen in new window to ensure good code quality and consistent formatting. You can check your code with one simple command:

npm run lint
1

I highly recommend installing the IDE extensions for these tools to see any errors during development.
